goesutils User's Guide
======================

Utilities for working with GOES files.


Getting Started
---------------

The best way to get started is to install (see below) and then explore the
examples gallery.


Installation
------------

`goesutils` is still in rapid development. You can get the latest release from
pypi via the command below.

.. code-block::

    pip install git+https://github.com/barronh/goesutils.git


Issues
------

If you're having any problems, open an issue on github.

https://github.com/barronh/goesutils/issues
