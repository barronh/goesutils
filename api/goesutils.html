goesutils.style module¶

goesutils.style.cmi2geocolor(ds)[source]¶

Convert CMI_01 (B), CMI_02 (R), and CMI_03 (G’) to color. Steps as follows:

  1. Clip all values to 0 to 1

  2. Raise all values to the power of (1 / 2.2)

  3. Replace G’ with G = 0.45 * R + 0.1 * G’ + 0.45 * B

  4. Clip all values to 0 to 1

Parameters:

ds (xarray.Dataset) – Expected to be a ABI-L2-MCMIPC file opened with xarray. Must have variables B=CMI_C01, R=CMI_C02, and G’=CMI_C03

Returns:

rgb – Array shaped (y, x, RGB) with values appropriate for pyplot.imshow

Return type:

numpy.array

Notes

Needs citation. If I recall, this formula came from goes2go
